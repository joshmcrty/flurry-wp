Add a falling snow effect to your WordPress site.

== Description ==

Flurry adds a falling snow effect to your WordPress site. It uses the Flurry jQuery plugin to create the effect. Flurry is an easy-to-use animated snow plugin for jQuery inspired by jSnow. It uses bullet characters as snowflakes, with no dependencies on images or CSS files.

== Installation ==

Upload the Flurry plugin to your blog and activate it.
